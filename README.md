# Basic-HTML-Website

This is a Basic HTML Website project for [roadmap.sh](https://roadmap.sh/projects/portfolio-website).

## Languages Used
- HTML 5
- CSS 3

## Key Skills
- Semantic HTML 5
- Multi-page website with proper styling
- Mobile adaptation using `@media (max-width:768px)`
- Contact form layout and styling

## How to Run
1. Download the Basic-HTML-Website folder
2. Ensure it contains: `home.html`, `project.html`, `articles.html`, `contact.html`
3. Open `index.html` in your preferred web browser
